# Guía de Usuario

## Acceso al Sistema

Abra su navegador y vaya a `http://localhost:8501`.

## Módulos

### 1. Equipos

- **Lista**: Visualice todos los equipos. Use los filtros superiores para buscar por categoría o ubicación.
- **Nuevo**: Registre nuevos activos. El código de inventario debe ser único.
- **Detalle**: Seleccione un equipo de la lista para ver su historial de movimientos y especificaciones.

### 2. Proveedores

- Gestione la información de contacto de sus proveedores.
- Registre contratos asociados para control de vigencia.

### 3. Mantenimiento

- **Programados**: Vea qué equipos necesitan atención próxima.
- **Nuevo**: Programe mantenimientos preventivos o correctivos.
- **Historial**: Consulte trabajos realizados y costos asociados.

### 4. Reportes

- **Dashboard**: Vista general del estado del parque informático.
- **Exportar**: Descargue reportes en PDF o Excel para presentaciones.

## Agentes Inteligentes

El sistema cuenta con agentes que monitorean constantemente:

- Mantenimientos próximos a vencer.
- Equipos obsoletos.
- Garantías por expirar.
- Costos excesivos de mantenimiento.

Las notificaciones aparecerán en la barra lateral izquierda.
